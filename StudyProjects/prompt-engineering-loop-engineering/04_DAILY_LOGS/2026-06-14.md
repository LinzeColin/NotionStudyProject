## Core Explanation Notes

### What is it?

Prompt Contract 是一次模型调用的输入输出协议。它把模糊意图拆成目标、上下文、输入、输出结构、约束、质量标准、失败模式和验证方式。

### Core theory

模型不是读心器。高质量 prompt 的关键不是堆长指令，而是减少歧义、选择正确上下文、定义可检查输出，并让失败可诊断。

### How should I use it?

每次给 AI 真实任务前，先写最小合同：

```text
Goal:
Context:
Input:
Output schema:
Quality bar:
Constraints:
Failure modes:
Verification:
Stop condition:
```

### How it helps life, decisions, learning, and work

它把 AI 从“聊天”变成“可委派任务”。你能更快判断模型失败在哪里，也能把经验沉淀成复用模板。

### ROI-max action

选 3 个你真实会交给 Codex/ChatGPT 的任务，各写一个 Prompt Contract，并跑 v1/v2 对比。
