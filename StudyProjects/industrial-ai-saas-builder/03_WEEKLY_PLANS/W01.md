## D01 Teaching Skeleton

### Active Recall

1. RAG 和模型记忆有什么区别？
2. 为什么工业 SaaS 需要可追溯证据？
3. 在回转窑/制造业场景里，哪些资料适合作为 RAG 检索源？
4. 什么情况下 RAG 不是正确工具？

### Core Explanation

RAG = retrieve evidence first, generate answer second. It is a decision architecture, not just a vector database.

### Decision Matrix

Compare:

- Model memory.
- RAG.
- Fine-tuning.
- Workflow automation.
- Agent execution.
- SQL/rules/simulation.

### Practice Scenarios

| Scenario | Expected judgment |
|---|---|
| Operator asks why an alarm happened and wants historical incident references | likely RAG + retrieval + citation + eval |
| System needs exact current kiln temperature and trend | not RAG first; use live data/query/monitoring |
| Repeated approval workflow for maintenance request | workflow automation first; RAG may support explanation |
| Expert wants the assistant to summarize changed SOP wording | RAG useful if sources are fresh and diff-aware |
