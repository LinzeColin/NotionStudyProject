# Study Preferences

## Known Preferences

- Default language: Chinese.
- Preferred output style: structured tables, explicit status, practical next actions.
- Notion content should be directly pasteable/importable and not overly decorative.
- GitHub should be the durable source of truth for study state.
- Notion should remain clean and useful, not cluttered with agent artifacts.
- High ROI and real usability matter more than broad feature coverage.

## Teaching Fit

Use:

- Active recall first.
- Short theory, then drills.
- Real scenarios from the user's current workflow.
- Daily scoring and dynamic adjustment.
- Friction logs.

Avoid:

- Generic motivational language.
- Huge shortcut lists without practice.
- Heavy Notion template systems before manual habits are stable.
- Claiming Top 1% from setup alone.

## Current Hypothesis

The most useful learning path is:

```text
operate faster -> classify better -> structure cleaner -> review daily -> simplify weekly
```
