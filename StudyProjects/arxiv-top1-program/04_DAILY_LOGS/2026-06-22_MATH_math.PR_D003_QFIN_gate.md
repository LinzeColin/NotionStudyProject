## Core Explanation Notes

### What is it?

Distribution 描述所有可能结果如何分布；variance 描述结果波动和不稳定程度；tail risk 描述低概率极端亏损是否会毁掉策略、账户或系统。

### Core theory

```text
posterior 只更新信念强度；
distribution 才描述收益/损失的完整形状；
action permission 需要 EV + variance + tail risk + cost + liquidity + rollback。
```

### Correction

教学中给出的 A/B 例子原称“两者 EV 差不多”，但数字不成立：

```text
A = 0.9*1 + 0.1*(-9) = 0
B = 0.55*2 + 0.45*(-1) = 0.65
```

因此 B 不只 variance/tail risk 更好，EV 也明显更好。已在教学中明确纠正。

### ROI Decision Rule

```text
宁愿研究中等胜率、亏损可控、分布稳定的机会，
也不要被高胜率但小概率巨亏的策略骗走。
```
