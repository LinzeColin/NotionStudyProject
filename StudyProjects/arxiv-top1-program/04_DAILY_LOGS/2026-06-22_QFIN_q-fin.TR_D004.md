## Concept Teaching

### What is it?

Adverse selection 指成交本身可能筛选出了更懂信息、更急于交易、或更有优势的对手方。对挂 limit order 的一方来说，成交不一定是好消息；如果成交后价格立刻朝自己不利方向移动，这笔 fill 可能来自 toxic flow。

### Core Mechanism

```text
limit order waits in book
informed/aggressive trader hits it
fill happens
mid/mark moves against the passive order
passive side realizes the fill was bad information
```

### Trading System Rule

```text
Fill is not success by itself.
Post-fill adverse movement is evidence.
If adverse movement is fast and large, reduce size, widen spread, cancel/reprice, pause signal, or require extra confirmation.
```

## Corrected Standard Expression

```text
我在 100 卖出后，市场中间价很快涨到 100.70，说明我可能卖早了、卖便宜了。
买方愿意立刻吃我的卖单，可能是因为他知道价格更应该往上走。
所以这笔成交可能是我作为 limit seller 被 informed buyer / toxic flow 打中。
```

## ROI Action Rule

For a trading agent:

```text
Do not optimize only fill rate.
Track post-fill adverse movement.
If fills are followed by adverse movement, lower passive size, widen quote, cancel stale orders, pause the signal, or require stronger confirmation.
```
